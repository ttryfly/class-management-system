# 培训班课时管理系统 (Class Management System)

一个基于现代化纯前端（HTML5 + CSS3 + Vanilla JS）和 Serverless 云端数据库（[Supabase](https://supabase.com/)）构建的全栈跨平台教务管理系统。专为中小型培训班、兴趣班设计，支持多机构/多教师在同一系统下数据安全隔离。

## 🌟 核心功能特性

*   **🔒 云端数据与账号隔离**
    *   接入 Supabase Email 鉴权，支持邮箱注册登录。
    *   基于 Supabase Row Level Security (RLS) 策略，实现不同机构/教师账号的数据物理级隔离，互不干扰。
*   **📊 实时运营看板**
    *   总资产统计（当前学员总数、开设课程总数）。
    *   一键预警：清晰列出剩余课时低于阈值（如 ≤3 节）的需续费学员。
    *   近期的消课签到动态流。
*   **👥 学员档案管理**
    *   学员基础信息录入与编辑。
    *   自动统计学员的**总累计购课量**和**当前可用剩余课时**。
    *   **一键导出 Excel/CSV 报表**，包揽该学员历史所有的充值记录与签到明细。
*   **📚 课程体系配置**
    *   灵活设置不同课程的“单次消课标准”（如：试听课扣 0.5 课时，正课扣 2 课时）。
*   **💰 充值与签到（资金/课时流转）**
    *   **购课充值**：详细记录充值时间节点和额度，余额实时到账。
    *   **自动签到消课**：选定学员与课程后，系统自动填入应扣课时完成抵扣。余额不足强制阻断。

## 🛠️ 技术栈描述

该项目抛弃了沉重复杂的传统全栈架构（Node.js/Spring等），将业务核心向前端与 BaaS 服务偏移。非常适合新手学习或者小机构快速无服务器部署。

*   **前端结构**: HTML5 + Vanilla JS
*   **前端样式**: 原生 CSS Variables 驱动的现代化与响应式定制 UI（适配移动端），使用 FontAwesome 字体图标。
*   **云架构底座 (BaaS)**: [Supabase](https://supabase.com/)
    *   Database: 建立在 PostgreSQL 基础上的云数据库。
    *   Auth: 邮箱验证登录机制。
    *   API: 采用 Supabase JS Client 进行零后端 CRUD 操作。

## 🚀 部署与本地运行建议

因为项目依赖于 Supabase 的 API 调用并在不同 `script` 间进行了模块拆分，**不能**直接双击 `index.html` 以 `file://` 协议运行（这会导致跨域和模块加载报错）。你需要一个简单的 HTTP 服务器。

### 1. 运行建库脚本 (数据库准备)
如果你想复刻或部署本项目，请先在你的 Supabase 项目 -> **SQL Editor** 中执行本项目内置的 `supabase_schema.sql` 脚本。它将为你一键建立所有需要的表结构和 RLS 安全规则。

### 2. 启动本地前端测试
打开终端，在项目根目录运行任意一种 Local Server：

```bash
# 如果你有安装 Node
npx serve .

# 如果你有安装 Python
python -m http.server 8000
```

随后打开浏览器访问 `http://localhost:8000` (或控制台提示的端口) 即可体验。
